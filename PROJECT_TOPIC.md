# Project Topic

## Қазақша

Қазіргі аутентификация және шифрлау әдістерін пайдалана отырып, қорғалған веб-қосымшаны әзірлеу.

## Орысша

Разработка защищённого веб-приложения с использованием современных методов аутентификации и шифрования.

## English

Developing a secure web application using current authentication and encryption methods.

## Жоба осы тақырыпқа қалай сәйкес келеді

Бұл жобада пайдаланушы аккаунтын қорғау және құжаттарды қауіпсіз сақтау функциялары біріктірілген.

Аутентификация бөлімі:

- Email арқылы тіркелу.
- Email verification code арқылы аккаунтты растау.
- JWT token арқылы қорғалған backend routes.
- 2FA арқылы қосымша қауіпсіздік деңгейі.
- Құпия сөзді қалпына келтіру механизмі.

Шифрлау бөлімі:

- Жүктелген файлдарды серверде AES-256-GCM алгоритмімен шифрланған түрде сақтау.
- Файлды preview, download немесе shared link арқылы ашқанда автоматты дешифрлау.
- `ENCRYPTION_KEY` арқылы шифрлау кілтін environment variable ретінде бөлек сақтау.
- Бұрын жүктелген plain файлдармен backward compatibility сақтау.

Қорғалған веб-қосымша бөлігі:

- Құжаттарды тек авторизацияланған пайдаланушы аша алады.
- Әр пайдаланушы тек өз құжаттарын көреді.
- Құжат әрекеттері activity log ретінде сақталады.
- Shared link уақытпен шектеледі.
